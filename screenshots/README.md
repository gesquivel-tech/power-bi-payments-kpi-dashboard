# Dashboard Screenshots

This folder contains exported screenshots from the Power BI Payments KPI Dashboard.

## Included Views

### `dashboard-overview.png`

Main executive dashboard page.

This view shows the core payment KPIs, including total payments, failed payments, failure rate, total amount, average payment amount, monthly payment trend, payments by status, failure rate by channel, and error distribution by type.

### `data-quality-operations-view.png`

Operational and data quality dashboard page.

This view focuses on pending payments, refunded payments, payments with error, error rate, SLA breaches, SLA breach rate, SLA status by channel, failed payments by region, and error distribution by type.

### `status-failed-filter-example.png`

Interactive filter example using the `Failed` status.

This screenshot demonstrates that the DAX measures respond correctly to visual interactions. When failed payments are selected, the dashboard recalculates the KPIs within that filter context.

### `region-argentina-filter-example.png`

Interactive filter example using `Region = Argentina`.

This screenshot demonstrates regional filtering and shows how payment volume, failed payments, failure rate, amount, and error analysis change for a selected region.

## Notes

All data shown in these screenshots comes from a synthetic dataset created for portfolio and demonstration purposes. No real customer, company, or payment data is included.
